# Netflix SQL + Power BI Dashboard Project

## Overview
This project analyzes Netflix dataset using MySQL and visualizes insights using Power BI.

## Tools Used
- MySQL
- Power BI

## Features
- IMDb analysis
- Genre analysis
- Country-wise content
- Viewership trends
- Time-based analysis

## Files
- Netflix_Dashboard.pbix
- netflix_queries.sql
- README.md

## Key Insights
- Drama is top genre
- USA has highest content
- High-rated shows are fewer but impactful
