# Netflix Power BI + MySQL Project
Power BI dashboard connected to MySQL database for Netflix data analysis.
## Tech Stack
- MySQL 8.0
- Power BI Desktop
## Files Included
1. netflix_db.sql - MySQL database backup
2. netflix on power bi.pbix - Power BI dashboard file
## How to Run This Project
1. MySQL mein database import karo:

2. 2. *Open Dashboard:* netflix on power bi.pbix file Power BI Desktop mein kholo
3. *Update Data Source:* Server = localhost, Database = netflix_db
4. *Refresh* dabao aur dashboard use karo

## Dashboard Features
- Top 10 Movies/Shows by Rating
- Content by Country Analysis  
- Genre-wise Distribution
- Yearly Release Trends
